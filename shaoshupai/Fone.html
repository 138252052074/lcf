少数派用户晋升指南

少数派

03月23日
9 1

欢迎来到少数派。

成立五年来，少数派已经不仅仅是「发现优质应用」，我们从应用延伸开来，正在围绕整个数字消费做内容尝试，我们期望用不限于文字的形式，帮助每个人更好运用数字产品或者科学方法，让生活和工作变得更好。

现在，你不仅可以在少数派阅读围绕数字生活的内容分享，也可以参与其中，成为内容创造者。如果你与少数派有着相同的理念，我们欢迎你的加入，成为少数派中的「少数派」。

用户身份

为了帮助你理解少数派内容创作体系，请允许我们先向你介绍少数派的 3 种用户身份：

🌕 普通用户
  • 对文章评论、点赞、收藏、分享；
  • 关注专栏、收藏专题；
  • 关注用户，以及向用户发送私信。

🌝 Matrix 作者

  • 在 Matrix 自由地写作；
  • 在 Matrix 发布的文章，可递向 Matrix 内的不同专栏投稿；
  • 成为专栏编辑，与主编一起管理和运营一个专栏；
  • 向自己管理的专栏投稿可获直接通过，无需等待审核；
  • 文章被编辑「精选」，在 Matrix 主页展示；
  • 文章有机会被推荐到少数派首页，在全站展示；
  • 不定期组织 Matrix 作者内部的有奖征稿活动。

🌚 签约作者

  • 文章可直接向少数派首页投稿，通过审核后即可在首页展示，并获得相应稿酬；
  • 在 Matrix 申请创建新专栏，审核通过后即成为该专栏的主编；
  • 每季度和年度的活跃作者会得到表彰，获得作者专属福利。

如何晋升

普通用户 → Matrix 作者

Matrix 是少数派的社区产品，你几乎每天都会在首页看到「Matrix 精选」文章，它们都来自 Matrix 作者的分享。如果你也想成为其中一员，填写 申请表格，告诉我们你想加入的理由，比如关注的领域、以往的文章、个人博客、微信公众号等，以及你打算在 Matrix 写什么，如果你的理由足够吸引我们，你就很可能成为 Matrix 作者。

这时，你便开启了少数派 Matrix 作者之旅，你可以在 Matrix 自由写作，分享你的文章。

Matrix 作者 → 签约作者

当你在 Matrix 分享了一些不错的文章后，若达到以下标准，你就可以申请成为少数派签约作者：

  • 你的文章里有 10 篇或以上获得 Matrix 精选
  • 你的文章有 3 篇文章或以上被推荐到少数派首页

少数派编辑部会综合评估你的文章和写作能力，再决定你是否可以成为，在 3 个工作日内给你答复。


如何保持签约作者身份

成为签约作者后，你需要：

  • 接受较高的文章修改成本和被拒稿的可能
  • 每季度（3 个月为一季度）发布的首页文章不能少于 5 篇

相应的也有一些小福利：

  • 每季度活跃度排名前 3 的作者，可获得神秘小礼品（活跃度：根据季度的文章数和文章受欢迎程度评定）

在一个季度结束时，如果你的活跃度未达要求，你的用户身份将会降回「Matrix 作者」。

我们期望你分享这些内容

新版之后，我们想挖掘更多关于「数字生活」的内容，所以选出了以下 6 个你们也会感兴趣的分类：效率工具、手机摄影、生活方式、游戏、硬件、人物。我们鼓励你围绕着这些话题，分享更多有意思的观点和经验。

效率

关于效率有着自己的看法、观点或技巧，知道如何正确地使用效率工具从而有效地提高效率，按自己的需求出发而不只是为了使用工具而本末倒置。

  • 经过 1700 个番茄时间，番茄工作法改变了我什么
  • 帮我打造 OmniFocus 工作流的得力助手：「模板脚本」的使用分享

手机摄影

希望你对手机摄影有一定的思考，知道手机摄影不仅仅只是美图+滤镜，了解相关的基础知识，有自己的摄影风格和拍摄技巧，接触过各类的手机摄影 App／了解这些 App 的使用技巧。

  • 用手机拍了一年照片后，这些技巧想要分享给大家
  • 想让你感受纯正「胶片模拟」效果的 VSCO X，138 元的订阅值不值？

生活方式

我们希望你对生活的理解不仅局限于「今天吃什么」而是对其有更广泛的经验和充满兴趣，除了使用工具和应用流程的经验之外，我们同样希望你能分享生活的其他维度。

  • 新的一年，通过记账完成自己的愿望清单吧
  • 数字生活 · 整理篇：我如何隐藏桌面系统凌乱的电线？

游戏

最好是个游戏老玩家，能信手拈来谈谈喜欢的作品。写一款游戏的时候不要就事论事，能够稍加列举同类对比，凸显其亮点。除了游戏的本身，我们希望你同时还是一个关注游戏幕后故事或开发团队的玩家。

  • 一场没有赢家的战争：This War of Mine 这是我的战争
  • Reigns，「朕已阅」的快感让人停不下来

硬件

热爱把玩新鲜的科技产品，你的文章不只是罗列相关的参数、堆砌好看的图片，更注重于它的实际用起来怎么样，以及思考它对你的工作生活带来的影响。

  • AirPods，无线耳机该有的样子
  • 借助树莓派与 HomeBridge ，将 YeeLight 彩光灯接入 Apple HomeKit

应用推荐、评测

你需要对某一类型或领域的 App 了如指掌，或对新 App 有很强的敏锐性，对一款 App 是好是坏有自己的判断标准，可以用不枯燥的方式向读者表达出来。我们更看重的是，你是否真的把 App 用于你的生活和工作学习中，而不仅仅是「玩」App。

  • 你负责脑洞大开，它负责呈现想法：思维导图 MindNode 评测
  • 深度测评：Tweetbot 何以如此特别

观点、技巧、教程

对某些类型的 App 或设备有深入研究，但能用浅显的语言表述出来，或传达自己的观点，或分享经验技巧，希望你的文章即使是小白级别的大众用户也能看懂。

  • 遭遇 iOS 日历广告？别急，可以这样解决
  • 我是如何折腾我的低配 Android 的

当然，好文章并不局限于以上这些例子，我们欢迎你分享在数字生活中的其他思考、观点或心得。

⚠️

如果你希望你的文章读起来更舒服，请你遵照少数派排版规范写作，详情见 少数派写作排版指南

如果你在写作期间遇到问题，你可以查阅 Matrix 作者 FAQ 和 编辑器使用指南
