什么是 Matrix

欢迎来到 Matrix。

现在，你不仅可以在少数派阅读围绕数字生活的内容分享，也可以参与其中，成为内容创造者。如果你与少数派有着相同的理念，我们欢迎你的加入，成为少数派中的「少数派」。

Matrix 是一个纯净、小众的写作分享社区。这里主张用工具改善生活，以写作促进思考，我们鼓励你写下有真实观点、实用价值且贴近生活的文少数派是一个致力于帮助每个人更好地运用数字产品或科学方法，让生活工作变得更好的内容平台。你不仅可以在这里阅读围绕数字生活的内容分享，也可以参与其中，成为少数派中的「少数派」 —— 分享数字生活的内容创造者。

成为作者

我们希望有更多人成为内容的创造者，但创造好内容也必定有门槛。为了维持 Matrix 的氛围和少数派的内容质量，Matrix 的写作权限将长期保持申请／邀请制。如果你也想成为其中一员，填写 申请表格，告诉我们你想加入的理由，比如关注的领域、以往的文章、个人博客、微信公众号等，以及你打算在 Matrix 写什么，如果你的理由足够吸引我们，你就很可能成为 Matrix 作者。

我可以在 Matrix 做什么

在 Matrix，你可以分享数字产品体验心得、你的数字生活点滴，或是你对数字生活、科技、互联网的思考……总而言之，这里欢迎你创造一切与数字生活相关的文章内容。

如果你写了一篇不错的文章，那么这篇文章有可能会被少数派编辑：

  • 选为 Matrix 精选
  • 收录到相应主题的 专栏
  • 被推荐到 少数派首页

如何开始在 Matrix 写作

你可以在 Matrix 内点击导航栏上的「铅笔」图标，进入编辑器写作界面，开始你的写作。如果你希望详细了解 Matrix 编辑器的使用，或在使用中遇到了问题，你可以查阅 Matrix 编辑器使用指南。

我可以在 Matrix 看到什么

  • 你可以在 Matrix 精选 页面看到由编辑精选的 Matrix 最新优秀文章；
  • 你可以在新推出的 Matrix 甄选 专栏看到 Matrix 每周的精彩好文；
  • 你也可以在 专栏广场 发现并关注感兴趣的专栏。

我在哪里可以看到我最新发布的文章

你在 Matrix 发布的文章默认只会出现在你的个人主页。当你的文章被「精选」、收录到「专栏」或被推荐到「少数派首页」后，则会在相应页面看到。

什么样的文章会被精选

我们每天会把符合条件的优秀文章推荐到「Matrix 精选」和「少数派首页」或收录到相应的「专栏」，让你的文章获得更多的曝光度，吸引更多同好的人与你讨论。

我们对文章精选的标准，包括但不限于：

  • 与数字生活、科技、互联网相关的内容；
  • 有自己的见解、观点，对读者也有一定的阅读价值；
  • 言之有物，不过于枯燥的表达方式；
  • 注重排版（附：少数派文案格式与风格规范）；
  • 无影响阅读体验的推广信息（如二维码、粗糙生硬的推广文案）；
    • 我们不喜欢通稿，但我们鼓励开发者／开发商在 Matrix 分享关于你们产品开发、运营的一切。为此，我们还特地开辟了 「开发者说」这个专栏。
      我们想看到你分享关于产品的开发思路、开发时的心路历程以及产品背后的理念。当你娓娓道来这个产品是如何从无到有，这就是对产品最好的宣传。
      同时，我们也欢迎你持续分享产品的使用技巧或教程，来帮助大家更好的了解和使用你的产品，这是留住并收获更多忠实用户最好的方法。
  • ……

什么样的文章可以被推荐到少数派首页

我们会不定期从 Matrix 精选文章中挑选优秀文章，推荐到少数派网站首页。我们挑选的基本标准包括但不限于：

  • 文章结构、思路上比较完整，而非帖子或笔记；
  • 行文用词上不要求非常书面，但不应太过口语化，或使用过多网络流行语；
  • 遵照少数派文案格式与风格规范，排版整洁、段落清晰、正确使用标点符号；
  • 内容有不错的参考或讨论价值；
  • 文章首发于少数派，并非是一稿多发的；
  • ……

发布到少数派网站的文章，自然会有更高的标准和要求。我们会在计划发布到网站首页之前，与作者联系沟通，并给出修改建议。

向专栏投稿

在不同的专栏里，你可以围绕着这个专栏的话题进行写作。如果你有不错的专栏策划，你可以把你的想法 通过邮件 或私信 官方人员 告诉我们，我们欢迎共同创建有趣的专栏。

在「发布成功提示页」投稿：文章成功发布后，你可以在下方的「推荐专栏」中自由选择投稿，也可以通过「搜索专栏」搜索其他专栏进行投稿。一篇文章可以同时投稿给多个专栏。

在「个人主页」投稿：在「个人主页 - 个人文章」中，点击选择你想投稿的文章卡片右上角「···」按钮，可以进行投稿。

在「文章详情页」投稿：在你想投稿的文章详情页点击导航栏的「操作」按钮，可以进行投稿。

在「专栏主页」投稿：在你想投稿的专栏页面内，点击文章右侧的专栏主编卡片上的按钮，可以进行投稿。

查看投稿进度

点击导航栏最右侧的头像，在下拉菜单中选择「投稿进度」，即可打开 #我的投稿 页面查看进度。

什么样的文章才会被专栏收录

为保证专栏质量，除了基础的文章主题符合该专栏，你的文章也应达到上「Matrix 精选」的标准，才能被专栏收录。

投稿文章未被专栏收录会怎么处理

投稿文章未被专栏收录不会对你的文章有任何影响，它仍然会在你的个人主页中，你也可以继续尝试向其它专栏投稿。

申请个人专栏

目前 Matrix 暂未开放个人专栏的申请。后续有此计划，敬请期待。

关注专栏有什么用

当你关注了某个专栏后，该专栏有新文章收录，你将会第一时间收到站内通知。

为什么我的文章被屏蔽了

如果你发布的文章与社区氛围不符，如明显软文倾向、带有不合规的推广链接等，我们很可能会对该文章进行屏蔽操作。文章被屏蔽后，文章浏览权限将变为「仅发布者可见」。

申请解除屏蔽

如果你对被屏蔽的文章进行了修改，内容符合 Matrix 社区 规范后，你可以私信 官方人员 申请解除文章屏蔽。

为什么我的文章被删除了

如果你的文章涉嫌抄袭他人、宣传盗版资源、进行人身攻击、破坏社区氛围或存在其它违反 Matrix 社区规范 的行为，你的文章将可能会被删除。

为什么我的账号被禁言／删除了

如果你在 Matrix 发布不友善的评论、广告、色情内容等违反 Matrix 社区规范 的评论／文章，我们会根据情节严重程度，对你进行 3～30 天甚至永久禁言处理，相关违规内容也会被屏蔽或删除处理。

如果你发现有未及时处理的违规内容或行为，或对官方的处理有任何异议，请私信联系 官方人员。

什么是私信

通过私信，你可以在少数派内与你喜爱的文章作者，或有共同兴趣爱好的读者直接一对一交流。

如果你不想使用私信功能，你可以在「个人头像 - 账号设置 - 消息与邮件」中选择关闭私信。

这篇文章我很喜欢怎么办

文章写得不错。
这篇文章太赞了！

当发出这样的感叹时，你可以点击文末的 ♥️ 图标来向作者传达你对这篇文章的喜爱；点击 📤 按钮可将文章分享到微博、微信或生成长图进行分享；点击 🔖 图标可收藏这篇文章。

除了访问网页，我还能在哪些地方看 Matrix

为了让你不错过 Matrix 的精彩文章，我们每周会以邮件形式给每位 Matrix 用户推送本周 Matrix 精选文章、专栏。

你也可以关注微博 @少数派Matrix、公众号 ID：sspai_matrix 和读读日报（少数派Matrix精选）来获取 Matrix 的精选文章和最新动向。

Matrix 是否有其他讨论交流的方式

你可添加微信号（ID : pz_sspai）为好友，把你的 Matrix ID 私聊给机器人，机器人会把你拉进 Matrix 用户交流群。

我们也有 Telegram 用户群，当你注册成功后，会收到一封邮件，邮件内附有 Telgram 用户群的链接，你只需要点击链接即可加入。

（注：Telegram 是一款跨平台的轻量即时通讯工具，登录需要科学上网。）

机器人是谁

机器人是一个缓慢运行在 Matrix 服务器上的发条机器人。🤖

我遇到了问题 / 我有一些建议，如何联系你们

Matrix 由每个身在其中的人共同组成，因此，每一位用户的建议对我们都至关重要，它可能会影响到 Matrix 产品和运营的未来走向。如果你对 Matrix 有任何意见或建议，欢迎私信 官方人员 告诉我们。
